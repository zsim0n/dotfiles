Zoltan's dotfiles
========

Developer . files pack for mac OS X. 

### Dependencies

* xCode
* brew
* wget, bash-completion, ngrep
* git-completion
* David Xu's port of posh-git for bash

### Content
* git configuration
	*aliases, gitignore for mac
* git prompt inspired by posh-git
* bash and git completion
* colourful shell
* aliases
